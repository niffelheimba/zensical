# Get started

## Windows Installation

- Download the latest Windows release.

[EXE Installer][EXE Installer]{ .md-button .md-button--primary }
    [EXE Installer]: https://pkgs.netbird.io/windows/x64
[MSI Installer][MSI Installer]{ .md-button .md-button--primary }
    [MSI Installer]: https://pkgs.netbird.io/windows/msi/x64

- Execute the installer and proceed with the installation steps.

- Open the desktop application via the desktop shortcut or searchbar.

- The default installation uses the Netbird's proprietary cloud management URL. To change the management URL to Northlake's URL do the following:

![Download Tutorial](https://github.com/niffelheimba/zensical/blob/main/images/netbirdtutorial.gif?raw=true){ align=left }
